Notes

If the program has some trouble understanding a particular quote, check for the following:

  • Did you use semicolons (;) to separate the scripture references, instead of colons (:) or commas?
  • Is there a single set of curly braces surrounding the scripture references? (e.g. {1 Cor 14:15} rather than {{1 Cor 14:15}}). There should only be one "{" and one "}" per quote.
  • Is the book abbreviation you're using the correct one? If not, quotes from the same book of the Bible will appear in different PDF files. Take a look at the approved abbreviations for each book.
